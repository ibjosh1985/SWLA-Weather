# SWLA Weather Alert

SWLA Weather Alert is a hardened, Python 3.10+ weather alert system for
AllStarLink and HamVOIP repeater nodes.

## Features
- NWS Warning alerts only
- Per-alert courtesy tones
- Audio playback with TTS fallback
- Alert expiration + clear tones
- Persistent alert history
- systemd service
- 24/7 safe operation
